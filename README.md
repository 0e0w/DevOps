# DevOps-开发和运营

本项目用来收集整理学习DevOps相关内容，包括DevOps的理念产品以及衍生的工具思路等。DevOps是一种方法，是将开发和运营紧密结合起来的方法。DevOps-Software development (Dev) and IT operations (Ops).
